== Description ==

Complete rewrite of previous version. I now have a much better UI to manage your maps and locations.

Built by The Marketing Mix Perth: http://www.marketingmix.com.au

== Installation ==

1) Install WordPress 4.0 or higher

2) Download the latest from: 

http://wordpress.org/extend/plugins/jquery-colorbox

http://wordpress.org/extend/plugins/my-mapped-locations

3) Login to WordPress admin, click on Plugins / Add New / Upload, then upload the zip file you just downloaded.

4) Activate the plugin.
